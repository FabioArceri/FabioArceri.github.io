Project "Beonbuy"
Find the shop closest to you which is selling what you are searching for
Catania, Beonbuy srls 2018 – now

Beonbuy is the project (the company) I'm currently working on (for). It is a sort of search engine that is intended to let you know where are located the closest shops to you selling a product of your interest. Beonbuy it's a distributed system made up of a mobile app, a web application, and a desktop syncing tool, that is able to potentially interoperate with any shop's management system in order to give always-updated information about stocks, prices, etc... Beonbuy is for helping you to reconnect to your local sellers and letting the local economy free to grow. Is Beonbuy the anti-Amazon? Maybe, or maybe not 😉.

As the Operations Manager I organize and supervise the tasks, and periodically plan out middle- and long-term roadmaps.

Project "Grypto"
Encrypt your data by using a binary file as a key
Catania, personal project 2016

I wrote Grypto as a part of an hackathon I was designing for a GDG event (actually, the name "Grypto" stands for GDG encrypter). Grypto takes two arguments: the file to be encrypted (that is the target) and a binary file (that is the key), such as an image, a sound file, an executable, etc... In encryption mode it works by sequentially looking at every single byte inside the target file and trying to map each of them inside the key file, also collecting the respective positions. While collecting the positions, it builds, byte after byte, the encrypted file as an appropriate sequence of the distances between the collected positions, until the last position is processed. On the other hand, when in decryption mode, Grypto reads the distances stored into the encrypted file and finally retrieve the respective bytes into the key file, thus reconstructing the original target file.
